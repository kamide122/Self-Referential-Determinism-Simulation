# Self-Referential Determinism (SRD) Simulation

This repository contains the official numerical simulation code for the framework of **Self-Referential Determinism (SRD)** in open quantum systems.

## Overview
This project models a two-level quantum system (qubit) interacting with an irreversible environment under non-Markovian dynamics. It demonstrates how an agent's **adaptive POVM measurement strategy** can actively suppress entropy production and accelerate the stabilization of classical pointer states, effectively out-competing environmental decoherence.

## How to Run
You can run this simulation completely free in your browser via Google Colab:
1. Install QuTiP: `pip install qutip`
2. Run `simulation.py`
